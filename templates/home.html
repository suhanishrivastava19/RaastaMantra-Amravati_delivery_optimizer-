What This System Offers

Powered by real Amravati road data and advanced AI algorithms

A* Algorithm
Heuristic-guided search finds the shortest path faster than brute-force. Uses Haversine distance as an admissible heuristic.
Uniform Cost Search
Explores routes in order of cumulative cost, guaranteeing the globally optimal (minimum cost) path.
Real City Map
Live Amravati road network from OpenStreetMap via OSMnx. Click any point on the real city map to set waypoints.
Analytics Dashboard
Compare algorithm performance with interactive Chart.js visualizations. Track efficiency, time, and cost metrics.

MongoDB Storage
All routes, users, and feedback stored persistently in MongoDB for historical analysis and improvement.
